## Description
雀和(Janho) は CocosCreator で制作したWebブラウザで動作する麻雀ゲームです。  
一人～四人で遊ぶことができます。(オンライン対戦)  
※四人に満たない場合は、CPUプレイヤーを追加して遊ぶことができます。  
  
牌譜保存機能などはありません。
